1.购买服务器

如果购买了域名要解析到你的服务器，要考虑是否要去备案。懒得备案的就去买海外的服务器供应商的产品，在大陆供应商购买的就需要去备个案了。

关于域名 的选择，国内注册的话，好像是不把域名的全部权限都给你的，这种情况下可能导致你的域名随时被国家收回。但是国外的话一般就不是这个样子。

买阿里云服务器之前，听说有更便宜腾讯云方案，学生机1块/月。但是稍微找了找没看到。目前10块/月的阿里云也可以接受吧！

另外购买阿里云服务器学生优惠需要经过一次学生认证。

2. 配置LAMP环境

在虚拟机安装的镜像需要换源

先敲下这两行代码

UPDATE: update is used to download package information from all configured sources.

UPGRADE: upgrade is used to install available upgrades of all packages currently installed on the system from the sources configured via sources.list

apt-get是某些linux发行版使用的一个“包管理器” 包管理器的作用是从源（Source）服务器下载最新的软件包，当你（apt-get install）的时候，就是从这个列表查询软件包的信息，添加到同时安装的列表里，然后下载安装。 这个软件包列表需要用户使用apt-get upate更新。 而upgrade则是对已安装的软件更新的过程。

接下来安装包

如果apt-get install 软件 失败

换成 sudo apt-get install

Apache2

Mysql

PHP7

重启Apache2和MySQL

测试php和apache2

保存后访问

ip(127.0.0.1或公网ip)/info.php

若有信息就说明成功，否则可能是缺少了某个PHP模块。

安装完成后 输入

sudo apt-get upgrade
