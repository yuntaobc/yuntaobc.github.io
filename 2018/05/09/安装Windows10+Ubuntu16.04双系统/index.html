Windows10 + Ubuntu16.04双系统

发表于 2018-05-09 | 分类于 Windows

安装过程

####准备：

  • Ubuntu 16.04镜像[link]
  • UltraISO[link]
  • U盘

安装过程

1. 下载镜像，并用UltraISO制作U盘启动盘，

打开UltralSO。
打开镜像（我之前的镜像丢了，还剩下 16.04.3版本做例子）。

点菜单栏的 启动 -> 写入硬盘映像，出现如下窗口。

格式化之后，写入并等待制作完成。

2. 设置BIOS关闭快速开机

不方便截图，但设置很简单

笔记本的关闭计算机的快速启动选项在这里，点开 更改当前不可用的设置，然后 关闭快速启动。

查一下自己开机进BIOS的方法，进入BIOS，调整启动顺序，把带U盘的那一项提前。保存后重启进入U盘引导界面。

3. 安装Ubuntu

重启后按F9到下面的菜单(修改过BIOS中启动顺序的无视这一步)

选择第二项直接安装，进入到下面的界面。

选择语言，下面有简中，我选的是英语。

选择是否连接网络（不连网就行）。

[此处应有图！！！]

下面这两项都不要选择，要不然安装时间会延长。

安装类型选择最下面的 其他Something else。

接下来，在硬盘空闲区域创建分区：

1.再点 + 创建分区，分配51200MB，选择主分区primary，文件系统默认为ext4（缩写），挂载到 / 。

2.再点 + 创建分区，分配至少256MB，选择主分区primary，文件系统默认为ext4（缩写），挂载到/boot。

3.点左下的 + 创建分区，分配8192MB，选择逻辑分区logical ，文件系统swap。

点击安装出来确认界面：

选择时区，键盘语言，用户密码，



等待安装完成后，拔出U盘重启。
