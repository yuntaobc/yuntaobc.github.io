Windows10远程桌面

发表于 2018-05-21 | 分类于 Windows

打算手机上安装微软远程桌面应用之后方便随时随地操作一下笔记本

操作环境：路由器，无线上网的笔记本

###0. 升级到Windows10专业版

Windows10家庭版不支持远程桌面 家庭版和专业版对比

按Windows + i 打开设置找到 更新和安全 -> 激活 -> 更改产品密钥 -> 输入专业版的密钥

系统会提示“升级Windows版本”，点击确认升级后，等待下载更新完成升级

没有密钥的话可以参考这里的方法：传送门

###1.设置允许远程连接

  • 打开控制面板(按 Windows + R 输入 control 回车)

  • 在小图标视图下打开 系统 选项

  • 点高级系统设置 -> 远程 -> 允许远程连接到此计算机 并在 仅允许运行使用网络级别身份验证的远程桌面 前面打勾

  • 按 Windows + i 打开设置找到 系统 -> 远程桌面 启用远程桌面

###2.更改电源计划从不睡眠(笔记本)

  • 打开控制面板电源选项 更改所有电源计划中的睡眠选项设置为 从不睡眠

###3.找到自己计算机的名称

  • 设置页面 系统 -> 关于 -> 设备名称

理论上到这一步就可以在Remote Desktop应用上添加你的计算机了(理论上大概是你的计算机直接接入公网，没有防火墙拦截远程连接等等)

但是连接时发现报错，找不到计算机 错误代码 0x104

####下面是一个简单粗暴的解决办法：

####0.添加路由器端口映射，配置花生壳内网穿透

  • 在浏览器输入 192.168.1.1 输入密码后，进入路由器管理界面

  • 打开虚拟服务器页面，如图添加内容:

  • 打开动态dns页面(或者是DDNS) 登录花生壳

  • 登录花生壳管理界面：传送门

  • 添加映射，填域名，内网IP，内网端口，外网端口用随机的就好

  • 下载花生壳客户端：传送门

  • 在PC端登录花生壳，保持后台运行

  • 尝试在remote desktop里添加你的电脑，PCname就填花生壳的域名附上添加映射的时候随机的外网端口号。类似这种格式(12224179q0.iok.la:12225)

  • 如果到这里还没解决的话，继续下面的操作吧

####1.关闭你的360安全卫士等防护软件，并在控制面板关闭Windows防火墙

####2.Windiws + R 输入 msconfig

  • 点开服务，隐藏所有 Microsoft服务，再点全部禁用，点确定，稍后重启

  • 点开启动 -> 点任务管理器的连接 -> 禁用所有不必要的应用(本机留下了VMware虚拟机 Realtek音频管理器)

  • 若要恢复，在msconfig 常规 点正常启动
